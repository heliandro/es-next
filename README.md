## ES NEXT

examples of next features of JavaScript.

#Requisites
* Nodejs v12.16.2+

#Instalation

for installation, run the following command:
> npm install

#How to use

for test an example, see how to execute:
> npx babel-node ./src/es2020/FILE-NAME.js
